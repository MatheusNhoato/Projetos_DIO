O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação padrão utilizada para criar e estruturar páginas na web. Por meio dele, organizamos todos os elementos que aparecem em um site, como títulos, parágrafos, imagens, links, listas e tabelas. O HTML funciona como a base de qualquer página, definindo a estrutura e o significado do conteúdo. Ele não é uma linguagem de programação, pois não executa lógica ou cálculos, mas sim descreve como as informações devem ser organizadas e exibidas pelo navegador.
