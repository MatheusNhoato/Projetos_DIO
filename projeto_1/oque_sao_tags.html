O que são tags?

Tags são os elementos fundamentais do HTML. Elas funcionam como rótulos que indicam ao navegador o tipo de conteúdo que está sendo exibido. Normalmente, uma tag possui uma abertura e um fechamento, envolvendo o conteúdo que ela descreve. Cada tag possui uma finalidade específica, como definir um título, um parágrafo, uma imagem ou um link, permitindo que o navegador interprete corretamente a estrutura da página.
