Tags de formatação

As tags de formatação permitem alterar a forma como o texto aparece na página, destacando partes importantes do conteúdo. Elas podem deixar palavras em negrito, itálico, sublinhado ou indicar trechos com maior relevância. Embora hoje o CSS seja mais utilizado para estilização, essas tags ainda são úteis para dar ênfase e significado a determinados trechos do texto. Exemplos de tags de formatação incluem negrito, itálico, sublinhado e destacado.
